Frequently Asked Questions

It is a long established fact that a reader will be distracted by the readable content of a page when looking at its layout.

> Layout is one dimension by design, it only works either one way (row/column).
> On the other hand Grid is a two dimension, it can work on rows and columns at the same time.

> Bootstrap is a component-based framework, it has pre-build ready to use components that can be use directly, those components can be edited by their build-in CSS also.
> On the other hand Tailwind is a CSS utility-first framework, it means their css can be use more like regular css, their components are not free to use like Bootstrap.

> CSS Box model is a like a map for html, that consists of: margins, borders, padding, and the actual content.

> Semantic Tag, more like tags, as there are multiple of them. There tags are used to define the meaning of the content they contain.
